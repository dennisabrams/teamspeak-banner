## Introduction

![ts3](https://user-images.githubusercontent.com/122950707/217850954-a10cedcd-ea12-46da-8eb3-12602d8991e0.png)

There are some great TeamSpeak 3 Banner Generators on GitHub, however I didn't find one with such much informations about the client & server side and the ability to customize the banner with an extern config file. 

My goal was to create a TeamSpeak 3 Banner with some important features:

### Program Explanation

The whole project was made with PHP & TS3 PHP Framework. The program connects to the TeamSpeak 3 server, authenticate and spawn an object for the virtual server to get Client sided informations and some Server sided informations. This is saved in variables and gets rendered on a image that is generated by PHP. The output is a png that should be used on the virtual ts server under Banner GFX URL. GFX should be set to 60 seconds because thats the miniumum TeamSpeak 3 supports. (Sadly the gfx timer on TeamSpeak is not 100% accurate so the time will be not synced on every second).
